* To run this files, First create 2 databases in PhpMyAdmin named "clumsy" and "eclumsy", then create table named "tusers" and "eusers" respectively, Bothe table has same 5 colums named "id", "firstName", "lastName", "email", "password".
* Create a main Folder and save all the html, php files and image folder in it.
* Store the create main folder in C/xampp/htdocs path.
* We can run the program by texting localhost/filename in any browser.
* We can see the database by texting localhost in any browser and go to MyAdmin.
